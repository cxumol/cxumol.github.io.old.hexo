word sucks

请您设想这样的场景：

普通少年·先驱·甲，由于工作需要，常常在电脑上打字。
小伙伴赵日天曾告诉先驱甲：“Word 功能又强，普及率广，甚至几乎每间办公室都离不开它。”
时间久了，先驱甲却发现，其实 word 的缺点还不少。
比如，鸡肋的自动保存；字体缺失、版本不一致引发校对混乱、格式混乱；再如.doc和.docx格式兼容问题；还有D版隐患、手在键盘鼠标间换来换去……

那一刻，先驱甲猛地拍桌子，铿锵有力，掷地有声：

或许、可能、应该、大概，我们写文稿，很大程度上，用不上 Word 辣么多七七八八功能。

纯文本主义

先驱甲同学，暂别复杂的 word ，拥抱了纯洁得像水一样的“记事本”
TA冲镜头邪魅一笑，秋波盈盈，道：“记事本儿小巧简单朴素环保易用速度快，谁用谁知道哇。”
其实，TA并没有壮烈地、轻易地、卸载掉 word 。
只不过是先用纯文本快速、流畅地写好后，再往 word 粘贴，接下来把需要的排版格式一一调整好

解释一下
简单来说，纯文本可以理解为，存在 txt 文件里的内容。它只包含文字本身，除了换行和标点，没有半点格式信息。

有个误区，以为.txt只能用记事本开。 事实上，文字相关软件基本都好使。比如写字板、word、手机小说阅读应用，甚至是浏览器……

所以，用纯文本写作最大好处，是查看和编辑起来毫无限制。

轻量标记语言，与高级文本编辑器

文章写长了，先驱甲总是遗忘哪里需要加上格式。这可不好:(
甲开动脑筋，灵感如闪电似利剑，炸响黑黢黢的夜空
TA把键盘上常见，实际少用的符号，或单独，或加以组合，作为记号。
哪里需要加粗，哪里需要斜体，咱就用记号给他标记上。
所以纯文本里又多了些“奇怪的”的标点

嗯，这个技巧，后人称之为“轻量标记语言”

先驱乙是先驱甲的好G友。
某天，先驱甲对先驱乙说：

用 word 给文章排版，十分合适；可若只是撰写文章 —— 或许会有更好的选择。
……

于是先驱乙也学会了这种技巧。
乙同学试了一下，愈发觉得真个是好办法。然后告诉了先驱丙丁
不过，先驱乙暗想：“虽然！记事本‘小巧简单朴素环保易用速度快’！但！不合吾装逼之道！”
TA上网查了一下，很快搞到功能强劲，比记事本好用N倍的高级文本编辑器——Notepad++

markdown 的诞生

密室。密谈。四人。
昏暗的白炽灯，投出苍劲而孤傲的背影，清冽冷峻。
先驱丙轻推桌沿，转椅回转，面向甲乙丁：

咱们虽然理念相同，每人用的具体标记却不一样。如果哪一回我想和乙共谱一章，岂不乱套？
不如让我们选出几个最最常用的格式，确定一下记号。先约定俗成。以后，就作为规范啦

先驱甲乙丁 见状兴奋，点点头，拍拍手：“好呀好呀！”
先驱甲有俗家姓名，叫“马克”。众人便将这种规范，命名为“马克裆”

先驱们到底是厉害！他们创立的规范，即使不经word补上格式，直接脑补也不费劲。
比如，*本段话被强调了*，> 啊，这段话是引用的。 ---若是独占一行，则为分割线
嘿嘿嘿，是不是一目了然？ 以下为分割线

至此，这种“标记法”算是诞生了。土鳖纯文本，摇身一变，成为格式精美、排版到位的高富帅文本，有两种途径：

  • 复制到 word ， 手动改
  • 脑补

先驱丁看到上面这两种方法，老觉得别扭。

为什么走到最后一步，还要手动加格式？脑补太low被忽视~555
告诉电脑『此符号对应彼格式』，让它机灵点，直接把格式改好，不好吗？

于是先驱丁坐镇实验室。
大门一关，痛饮脑X金一公升，夜以继日的钻研黑魔法咒文和炼金阵图。
三天三夜，划时代发明——“马克裆牌渲染器” 闪亮登场

渲染器这家伙，就像烤面包机一样：吃下去的，是带“标记语言”迪奥斯文本；拉挤出来，却是香喷喷的、亮闪闪的高富帅文本。

终于等到这一天。Markdown 脱离了 Word 的襁褓，迎来新生，成为了独立的存在。

包括本文在内，此博客所有文章既由此而成。

不装逼的 markdown

时移世易，沧海桑田。先驱人已殁，节操代代传。

可惜世风日下，人心不古

202X年。许多人掌握 markdown ，自觉逼格高人一等，常常对只会使 word 的人，投之以不屑的目光。宇宙装逼大赛组委会，也将会 markdown 列为加分项。markdown俨然吹捧成了神技。

渐渐，被捧上神坛的 markdown ，成为高贵冷艳的象征之一。许多人徒闻其名。更多人匆匆瞥一眼它的名字，便很快走开，喃喃：“我还是食人间烟火吧”。打消了对 markdown 的兴趣。

……

又是很多年过去……

太阳依旧挂在晴朗的天空。命运齿轮咔咔。一位名叫『贾区先』的少年，来到自家后院，拂去岁月的尘土，找回失落的宝藏。

那是一张泛黄的16开作业本纸。密密麻麻的标记语法，早已被岁月蚕蚀，模糊不清。
翻到反面，几行字却仍然清晰醒目

Markdown 设计哲学：
1. 专注：实现形式与内容的分离
2. 简约：用最简的语法达到效果
2. 直观：符合人直觉的表达方式
—— 先驱甲 留
